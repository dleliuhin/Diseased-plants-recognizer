function writeXlsxGroupData(iTableFileName, ...
                            iLeaveGroup, ...
                            iStartColumn, ...
                            iStartRow)

WRITEXLSXGROUPDATA

Write current plantic iLeaveGroup statistic for each image from this group to table in Result/Tables/iTableFileName.xlsx from specifics iStartColumn and iStartRow.

  • Syntax
%	[] = WRITEXLSXGROUPDATA(ITABLEFILENAME,
%                           ILEAVEGROUP,
%                           ISTARTCOLUMN,
%                           ISTARTROW)
%
% * Input
%
% -- iTableFileName - name of the file in which need to write averages.
%
% -- iLeaveGroup - structure with healthy and diseased parts of a plants.
%
% -- iStartColumn - column in table of iTableFileName from which need to
%                   write data statistic.
%
% -- iStartRow - row in table of iTableFileName from which need to
%                write data statistic.
